# Transformer Parallelism Page

This folder contains a static GitHub Pages site with a diagram-first overview
of distributed compute in transformer training.

## Quick start

1. Create a new GitHub repository.
2. Copy the contents of this folder into the repo root.
3. Commit and push.
4. In the repository settings, enable GitHub Pages for the `main` branch and
   the root directory.

## Files

- `index.html`: Main page content.
- `styles.css`: Typography, layout, and diagram styling.

## Customization

- Replace or extend the SVG diagrams in `index.html`.
- Update the author name and date in the hero section.
- Add more sections if you want to cover pipeline parallelism or optimizer
  states.
